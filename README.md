# UP Token Tap Game - Static Build

This is a pre-built static version of the UP Token Tap Game, ready to deploy to Vercel without any build process.

## Quick Deploy to Vercel

### Method 1: Drag & Drop (Easiest!)

1. **Zip this entire folder**
2. **Go to**: https://vercel.com/new
3. **Drag and drop** the ZIP file
4. **Click "Deploy"**
5. **Done!** No build configuration needed

### Method 2: Vercel CLI

```bash
cd tap-game-static
vercel --prod
```

That's it! The game is already built and ready to serve.
